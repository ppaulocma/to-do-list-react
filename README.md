Projeto Lista de Tarefas (To-Do List)

Este projeto consiste em uma aplicação de lista de tarefas (To-Do List) desenvolvida utilizando React no frontend, PHP no backend e MySQL como banco de dados, seguindo uma arquitetura MVC simples e Padão PSR-4. A aplicação permite adicionar, editar, excluir, marcar como concluída ou pendente e visualizar tarefas.

Bibliotecas Utilizadas no Frontend
    - bootstrap
    - ReactJS
